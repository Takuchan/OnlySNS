# OnlySNS

自分専用の学習SNSです。投稿・検索・分析に加え、ローカルLLM(Ollama)を使ったAI補助機能を利用できます。

## 起動方法 (Docker Compose)

```bash
docker compose up -d --build
```

起動サービス:
- `postgres`: データベース
- `ollama`: ローカルLLMサーバー (`127.0.0.1:11434`)
- `backend`: Go API (`127.0.0.1:8080`)
- `frontend`: Next.js (`127.0.0.1:3000`)

`backend` は起動時に `OLLAMA_MODEL` / `OLLAMA_EMBEDDING_MODEL` を自動取得します。

## AI機能 (ローカル実行)

- AI学習相棒のつっこみ
- 反応
- 4択クイズ生成
- 関連投稿の推薦

上記は全てバックエンドから Ollama API を呼び出して実行されます。

## キャラクター成長ロジック (内部仕様)

`Your Learning Journey` のステータスは、投稿ごとの単純な段階登録ではなく、全投稿の内容を集計して再計算する方式です。

- 判定タイミング:
	- キャラクター状態取得時に、投稿データ全体から都度算出
- 評価対象:
	- 投稿本文の情報量
	- タグの多様性と継続性
	- コードスニペットやメディアの活用
	- 反応値 (いいね/シェア)
	- 学習継続日数に基づく一貫性
- 成長方式:
	- 上記シグナルを合算して学習ポイントを生成
	- 学習ポイント帯に応じてステージを決定

### 非公開ポリシー

ユーザー体験上、成長の詳細条件 (しきい値・計算式) はUI上に表示しません。
このため、フロントエンドでは投稿数や進捗バーの閾値を明示せず、「学習の積み上げで成長中」という表現のみを表示します。
